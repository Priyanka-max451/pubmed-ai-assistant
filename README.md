# 🧠 PubMed AI Research Paper Assistant

An AI-powered application that fetches real-time medical research papers from PubMed and provides intelligent summaries and question-answering using LLM.

---

## 🚀 Features

- 🔍 Fetch real research papers using PubMed API
- 🧠 AI-based summarization using LLaMA3 (Ollama)
- 💬 Ask questions about research papers
- 📖 View abstracts interactively
- ⚡ Optimized for performance using caching and prompt tuning

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- Ollama (LLaMA3)
- Biopython (Entrez API)
